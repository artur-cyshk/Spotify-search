## Spotify-search ##
__Visit web-site of this application:__ *https://spotify-searcher.herokuapp.com*
*This application allows user to sign in, search and get music from spotify.*

__* To start this project__:
  * Git clone;
  * cd <repo_name>;
  * npm i;
  * npm start.
